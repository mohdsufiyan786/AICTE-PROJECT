# AICTE-PROJECT
This project for aicte internship
NoteVerse is a centralized web application designed for students to upload, share, and manage academic notes. It features an AI-powered Summarizer and "Chat with PDF" functionality to help students extract key insights instantly from lengthy documents.

Key Features
Note Sharing: Upload and categorize academic notes for peer collaboration.


AI Summarization: Get concise bullet points from long PDFs using NLP models.
+1


Chat with PDF: Interact with your documents to ask specific questions.

Secure Auth: User authentication and real-time data management via Firebase.

Tech Stack

Frontend: HTML, CSS, JavaScript (React).

Backend: Firebase (Authentication & Storage).


AI Engine: Python, Hugging Face API, and Streamlit.


Machine Learning: Scikit-learn for data processing.

Installation
Clone the repo: git clone 

Install dependencies: pip install -r requirements.txt

Run the app: streamlit run app.py 
+1

Future Scope
OCR integration for handwritten note recognition.

Voice-to-text summaries.

Multi-language support.
